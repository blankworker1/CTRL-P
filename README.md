CTRL–P is a time-governed printing protocol operating at street level.

It is composed of five structural components:

**Clock** — a public, deterministic 5-state slow clock anchored to a fixed UTC epoch.

**Nodes** — physical installations that replace one panel every 144 hours.

**Cycles** — 30-day rotations defined entirely by elapsed time.

**Feed** — a continuous, hourly image ledger from each Node.

**Interface** — a minimal public layer that renders visibility without influencing operation.

The system is deterministic and non-reactive.

Time governs action.

Replacement marks progression.

Continuity establishes presence.

The protocol does not depend on audience, interpretation, or adjustment.

It persists through structured repetition.
